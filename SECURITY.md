# Security Policy

If you discover a security vulnerability in this repository, please report it privately so we can respond and fix it before public disclosure.

Preferred reporting methods
- GitHub Security Advisories: Use GitHub's private security advisory flow for this repository.
- Email: oprl53n2s@mozmail.com (for private/sensitive reports)

What to include
- A clear description of the vulnerability and steps to reproduce it.
- The affected distro(s) and version(s) and any relevant test or script names.
- Any logs, traces, or proof-of-concept code that demonstrates the issue.

What we will do
- Acknowledge receipt within 3 business days.
- Triage and assign a maintainer to investigate.
- Work to provide a fix or mitigation and coordinate disclosure with the reporter.

Notes
- Do NOT open a public issue to report a vulnerability. Use the private reporting methods above.
- If you need to send encrypted information, contact the email address above and we will provide instructions (PGP key) if needed.
